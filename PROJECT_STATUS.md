## 🔧 技术架构

### 前端技术栈：
- **框架**: Next.js 14 (App Router)
- **语言**: TypeScript 5
- **样式**: Tailwind CSS 3 + 自定义设计系统
- **动画**: Framer Motion 10
- **图标**: Lucide React + React Icons
- **工具**: ESLint, PostCSS, Autoprefixer

### 项目结构：
```
zen-culture-website/
├── app/                    # 页面路由 (App Router)
│   ├── page.tsx           # 首页
│   ├── layout.tsx         # 根布局
│   ├── loading.tsx        # 加载组件
│   ├── globals.css        # 全局样式
│   └── ceramic-art/       # 陶瓷艺术页面
├── components/            # 可复用组件 (12个)
│   ├── Navbar.tsx         # 响应式导航
│   ├── Footer.tsx         # 页脚
│   ├── HeroSection.tsx    # 英雄区域
│   ├── FeatureCards.tsx   # 功能卡片
│   ├── GalleryPreview.tsx # 交互式画廊
│   ├── UpcomingEvents.tsx # 活动展示
│   └── Testimonials.tsx   # 评价轮播
├── public/                # 静态资源
├── package.json          # 项目配置
├── next.config.js        # Next.js配置
├── tailwind.config.js    # Tailwind配置
├── tsconfig.json         # TypeScript配置
└── vercel.json           # Vercel部署配置
```
